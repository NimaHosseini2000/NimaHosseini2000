## About Me

I'm an AI/ML Engineer focused on building intelligent systems that combine machine learning fundamentals with modern generative AI applications.

My work spans across Natural Language Processing, Retrieval-Augmented Generation (RAG), sentiment analysis, and machine learning algorithm development. I enjoy understanding how models work under the hood, which is why I often build algorithms from scratch before leveraging production-grade frameworks.

### Areas of Interest
- Retrieval-Augmented Generation (RAG)
- Large Language Models (LLMs)
- Natural Language Processing (NLP)
- Machine Learning Engineering
- MLOps & AI Applications
- Data Analysis & Predictive Modeling

### Recent Projects
🔹 **RAG Chat Bot** – Developed an AI-powered chatbot leveraging Retrieval-Augmented Generation techniques to deliver context-aware and knowledge-grounded responses.

🔹 **Sentiment Analysis** – Built NLP pipelines for sentiment classification and text analysis, exploring how machine learning can extract meaningful insights from unstructured data.

🔹 **Linear Regression From Scratch** – Implemented core machine learning algorithms without relying on high-level libraries to strengthen understanding of optimization, gradient descent, and model training.

### Philosophy
I believe the best AI engineers understand both the theory and the systems behind machine learning. My goal is to build AI products that are not only intelligent but also reliable, explainable, and impactful.

Currently exploring advanced LLM applications, vector databases, agentic AI systems, and scalable machine learning architectures.
